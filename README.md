# NoNameNews - News Hub Pro

A modern, AI-powered news aggregation platform built with React, Supabase, and TypeScript.

## Features
- **Automated News Fetching:** Crawls RSS feeds every 15 minutes.
- **AI Processing:** Uses Groq (Llama 3) to rewrite and summarize articles.
- **AI Images:** Generates unique visuals for every story using Pollinations.ai.
- **Multilingual:** Live translation support.
- **Admin Dashboard:** Trigger news updates and manage content.

## Tech Stack
- Frontend: Vite, React, Tailwind CSS, shadcn-ui
- Backend: Supabase (Auth, Database, Edge Functions)
- Automation: GitHub Actions

## Setup
1. Clone the repo
2. `npm install`
3. `npm run dev`
